What is float?

The float property in CSS allows you to position an element to the left or right of its container, letting content flow around it. It was widely used for layouts before Flexbox and Grid became popular.

Basic example

Here is an example of two boxes floating side by side:

Left Box
float: left;
Right Box
float: left; 
.left-box {
  float: left;
  width: 45%;
}
.right-box {
  float: left;
  width: 45%;
}

Pros and Cons

  • Pros: Easy to use for simple layouts, widely supported.
  • Cons: Can cause overflow issues, requires clearfix, less flexible than Flexbox or Grid.

Usage Tips

  • Use overflow: hidden or the .clearfix technique to avoid container collapse issues.
  • For new projects, consider Flexbox or Grid for more complex layouts.
