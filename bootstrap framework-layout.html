How CSS Framework Layouts Work

CSS frameworks like Bootstrap provide a responsive grid system and prebuilt components. The grid uses rows and columns to align content, adapting to different screen sizes automatically. 

<div class="row">
  <nav class="col-3">Navigation</nav>
  <main class="col-6">Main Content</main>
  <aside class="col-3">Sidebar</aside>
</div>

Bootstrap Grid Example CSS: 

.row {
  display: flex;
  flex-wrap: wrap;
}
.col-3 { flex: 0 0 25%; }
.col-6 { flex: 0 0 50%; }
@media (max-width: 700px) {
  .col-3, .col-6 { flex: 0 0 100%; }
}
