How CSS Flexbox Layouts Work

display: flex enables flexible, one-dimensional layouts. Flexbox was designed to align, distribute, and order items easily within a container, making it ideal for modern responsive design. 

<div class="container" style="display: flex;">
  <div class="nav">Navigation</div>
  <div class="main">Main Content</div>
  <div class="sidebar">Sidebar</div>
</div>

Flexbox Example CSS: 

.container {
  display: flex;
}
.nav, .sidebar {
  width: 22%;
}
.main {
  flex: 1 1 0;
}
