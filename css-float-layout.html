How CSS Float Layouts Work

The float property was originally designed to wrap text around images. Early web developers repurposed it to create multi-column layouts by floating <div> elements left or right. To prevent parent containers from collapsing, a clearfix technique is used. 

<div class="container">
  <div class="nav" style="float:left; width:25%;">Navigation</div>
  <div class="main" style="float:right; width:50%;">Main Content</div>
  <div class="footer" style="clear:both;">Footer</div>
</div>

Clearfix Example: 

.container::after {
  content: "";
  display: table;
  clear: both;
}
